1 _ Ajouter un "config.json"
  
  Ajouter un fichier "config.json" avec à l'intérieur les informations de connexion sous la forme suivante :
  {
    "user": "YOUR_USER",
    "host": "YOUR_HOST",
    "password": "YOUR_PASSWORD",
    "database": "YOUR_DB_NAME"
   }
